Hi，我是育晟，這裡是一個side project，專案功能包括

  1. 完整會員系統-含第三方登入支援(Gmail, Facebook)
  2. 呈現簡易股票基本面、籌碼面數據和圖表
  3. 透過數據進行股票篩選和輸出篩選結果至.csv
股票檔數共50檔，為2021年4月時元大台灣50的成份股。

專案使用技術包括
區塊 內容
前端 HTML, CSS, Javascript, jQuery, Bootstrap
後端 Django
資料庫 PostgreSQL
佈署 Docker, GCP(Cloud Run, Cloud SQL, Cloud Storage, Secret Manager)
